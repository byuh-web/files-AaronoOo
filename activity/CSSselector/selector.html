What is the difference between an ID and a Class?

ID is used for one unique element (#), while Class is used for multiple elements (.).

What is the difference between a SPAN tag and a DIV tag?

< span> is inline (stays in the same line), < div> is block (starts on a new line)

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

< div> is a generic container with no meaning — just used for layout or grouping things. < header>, < main>, < footer> are semantic tags — they describe the role of the content (top of page, main content, bottom of page).

What are the different ways to select a color?

You can choose colors in CSS using color names, hex codes, RGB/RGBA, or HSL/HSLA values.
